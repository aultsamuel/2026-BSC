Tiebreaker Rules

  1. Head-to-head record — among only the tied teams.
  2. Head-to-head point differential — net points in games between the tied teams.
  3. Overall point differential — net points across all regular-season games (DIFF column).
  4. Points scored — total offensive output in regular-season play.
  5. Commissioner's decision — if all above are still equal.
